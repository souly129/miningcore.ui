# MeowPool WebUI

A modern, themeable dashboard for MiningCore designed for real-time mining pool monitoring with a clean and responsive interface.

Built with a focus on clarity, performance, and a soft visual identity inspired by calico color palettes.

Live display at https://meowpool.net/

---

## Features

- Real-time pool and miner statistics
- Clean card-based dashboard layout
- Worker, pool, and hashrate monitoring
- Solo and pool mode tab switcher
- Clear and easy to read connection information
- Responsive design for desktop and mobile
- Subtle hover interactions and visual feedback
- Fully themeable UI system
- Multiple visual skins including Calico, Void Cat, and Neon Tabby

---

## Themes

The UI supports multiple selectable skins:

### Calico (Default)
A warm light theme with cream backgrounds and orange accents.

### Void Cat
A dark minimal theme with high contrast styling and subtle glow effects.

### Neon Tabby
A cyberpunk inspired theme with neon cyan, magenta, and glowing UI elements.

Themes can be switched through the UI dropdown menu.

---

## UI Overview

The interface is designed around a modular dashboard layout:

- Coin and pool overview cards
- Real-time hashrate tracking
- Worker status monitoring
- Clean typography and spacing system
- Fast and minimal navigation

---

## Configuration

This UI connects directly to a Miningcore backend via API.

Ensure your Miningcore instance is properly configured and accessible before use.
